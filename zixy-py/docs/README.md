# zixy-docs

This repository contains documentation for the zixy project


## Building the docs locally

Requirements:

1. The [just command runner](https://just.systems/man/en/packages.html) (recommended)
2. [uv](https://docs.astral.sh/uv/getting-started/installation/)
3. [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) (optional)


### Clone the repo

First clone the repository

```shell
git clone git@github.com:quantinuum/zixy.git
```


### Build the sphinx docs

Next, navigate to the `zixy-py/docs` directory and run

```shell
just build
```

This will build the html pages for the API docs and example pages.

To delete all of the files generated by the build

```shell
just clean
```

### Serve the html locally (optional)

```shell
just serve
```

Requires npm installation.
